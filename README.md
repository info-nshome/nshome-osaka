# NS HOME コーポレートサイト（LP）

## プロジェクト概要
大阪の住宅リフォーム会社「株式会社 NS HOME」の新規顧客獲得を目的としたコーポレートサイト（ランディングページ）です。

## 完成済み機能

### ページ構成

#### メインページ（index.html）
1. **トップ（ヒーロー）** - メインビジュアル + キャッチコピー + CTA
2. **代表メッセージ** - 代表 大串 修登のごあいさつ
3. **一般のお客様へ** - 住宅リフォーム / 水回り / 内装 / リノベーション
4. **不動産・管理会社様へ** - 原状回復 / 空室対策 / 賃貸リフォーム / リノベーション提案
5. **店舗・事業者様へ** - 店舗改装 / 開業前の内装相談 / レイアウト変更（施工写真は準備中）
6. **NS HOMEが選ばれる理由** - 4つの強み
7. **ご相談の流れ** - 6ステップのタイムライン
8. **対応エリア** - 関西一円（大阪/京都/兵庫/滋賀/奈良/和歌山）
9. **会社案内** - 会社情報テーブル
10. **お問い合わせ** - 電話 / 公式LINE / メール / フォーム
11. **CTA** - お問い合わせ誘導
12. **フッター** - 会社情報 + ナビゲーション

#### 施工事例一覧ページ（works/index.html）
- カード型グリッドレイアウト（PC: 3列 / タブレット: 2列 / スマホ: 1列）
- **フィルタ機能**：すべて / 水回り / 内装 / リノベーション / 原状回復 / 店舗
- カードクリックで個別ページへ遷移
- CTA（お問い合わせ誘導）付き

#### 施工事例 個別ページ
- パンくずリスト
- タイトル（エリア含む）+ 工事の種類タグ
- 概要テーブル（工事内容 / エリア / 費用目安 / 工期）
- **ビフォー・アフター写真**（Before→After 1組）
- **施工の流れ**（全写真をグリッド表示、説明文なし）
- 工事内容（本文）
- 一覧に戻るボタン
- CTA

**公開中の施工事例：**
1. 兵庫県｜倉庫内トイレ新設工事（水回り）— `works/toilet-installation.html`
2. 大阪市内の事務所｜ウッドデッキ新設（その他）— `works/wooddeck.html`
3. 大阪市｜クロス張り替え工事（内装）— `works/crosswall.html`
4. 大阪市｜事務所内トイレ 天井ボード張り替え（内装）— `works/ceiling-board.html`

## 施工事例の追加手順

### 個別ページの追加
1. 既存の施工事例ページをコピーして新しいファイル名にする（例: `works/bathroom-reform.html`）
2. タイトル、description、canonical、本文、画像パスを書き換える
3. ビフォーアフター写真は `images/` に保存し、ページ内の画像パスを設定する
4. `sitemap.xml` に拡張子なしの正規URLを追加する

### 一覧ページへのカード追加
1. `works/index.html` を開く
2. `<!-- ▼▼▼ 事例カード ここから ▼▼▼ -->` ブロックをコピー
3. `<!-- ★ 新しい事例カードを追加する場合は〜 -->` の場所に貼り付け
4. `data-category`、`href`、サムネイル画像、タイトル、メタ情報を書き換え

### フィルタのカテゴリ
| カテゴリ名 | data-category 値 |
|-----------|-----------------|
| 水回り | `水回り` |
| 内装 | `内装` |
| リノベーション | `リノベーション` |
| 原状回復 | `原状回復` |
| 店舗 | `店舗` |

※ 複数カテゴリの場合はカンマ区切り: `data-category="水回り,内装"`
